Starting a chat

You can start a chat with someone in your Contact List.

About this task

To start a chat with someone in your Contact List:

Procedure

  1. Highlight the name of the contact in your Contact List. For tablets, the contact's Person Card will display. For phones, choose the Select action to display the contact's Person Card.
  2. The Person Card displays the Chat tab by default. Type your message and then select the Send action.

Note: If the contact is offline, the Send action will be grayed out and unusable.

  1. Look for a response from your chat partner in the chat window.
  2. To close the chat window when you are done chatting on a tablet, select Close Chat from the action menu. For phones, select the Back button to return to the previous Connections Chat screen.
