Creating groups

You can organize your contacts by placing them into groups. You can create a group when adding a contact.

About this task

To create a group:

Procedure

  1. From your Contact List, press the Menu key and select the Add action.
  2. Select the Add Group action. The Add Group screen opens.
  3. Select either the Private Group or Public Group action.
  4. Enter a name for the new group and select the Add action.

Results

You can now add more contacts to the group.
